This project is a web-based calculator built using React, which provides basic arithmetic operations and follows the user stories provided by FreeCodeCamp.

You can access the live version of this project [here](https://f9x9f2.csb.app/).

## Features

- Addition, subtraction, multiplication, and division of numbers
- Decimal point support for floating-point numbers
- Clear button to reset the calculator to its initialized state
- Responsive design for various screen sizes
- Supports chaining multiple operations

## User Stories Implemented

1. The calculator contains an equal (=) button.
2. There are clickable number buttons (0-9).
3. Mathematical operator buttons (+, -, *, /).
4. A decimal point (.) button is available.
5. A clear button (C) resets the calculator.
6. Display is used to show input and results.
7. The clear button resets the calculator to its initialized state.
8. Input numbers are displayed."
9. The calculator performs addition, subtraction, multiplication, and division operations.
10. Numbers cannot start with multiple zeros.
11. Decimal points are handled correctly.
12. Decimal numbers can be used in operations.
13. Consecutive operators use the last entered operator.
14. Operators following the equals (=) sign start a new calculation.

## Installation and Usage

1. Clone the repository: `git clone https://github.com/abhisheksharma1310/javascript-calculator.git`
2. Navigate to the project directory: `cd javascript-calculator`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open your browser and go to `http://localhost:3000` to access the calculator.
